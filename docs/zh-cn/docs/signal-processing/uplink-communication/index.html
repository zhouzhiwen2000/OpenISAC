上行通信

上行使用与下行相同的 NNN、NCPN_\mathrm{CP}NCP​、Δf\Delta fΔf、帧内符号索引 mmm 和导频位置，但采用独立的发送资源网格 BγUL\boldsymbol B_\gamma^\mathrm{UL}BγUL​ 与 ZC 根序列。UE 已通过下行链路完成与 BS 的初始同步；上行是一条完整的 UE→BS 通信链路，而不是重新执行一套独立初始捕获或简单反向重放下行数据。TDD 中，上行资源网格在 m∈SDL∪SGm\in\mathcal S_\mathrm{DL}\cup\mathcal S_\mathrm{G}m∈SDL​∪SG​ 时置零，仅在 m∈SULm\in\mathcal S_\mathrm{UL}m∈SUL​ 时承载上行符号。

BS 侧频域模型

Section titled “BS 侧频域模型”

上行 ZC 的 LS 估计为

H^n,0,γUL,LS=Yn,0,γULznUL.\hat H_{n,0,\gamma}^\mathrm{UL,LS} =\frac{Y_{n,0,\gamma}^\mathrm{UL}}{z_n^\mathrm{UL}}.H^n,0,γUL,LS​=znUL​Yn,0,γUL​​.

将其变换到时延域后，可依据循环前缀范围限制有效抽头并进行 Wiener 平滑，再变回频域。这样既保留多径结构，又降低全带宽 LS 估计中的噪声。

均衡与信息恢复

Section titled “均衡与信息恢复”

使用 ZF 或 MMSE 系数 Gn,mULG_{n,m}^\mathrm{UL}Gn,mUL​ 后，

d^n,m,γUL=Gn,mULYn,m,γUL.\hat d_{n,m,\gamma}^\mathrm{UL} =G_{n,m}^\mathrm{UL}Y_{n,m,\gamma}^\mathrm{UL}.d^n,m,γUL​=Gn,mUL​Yn,m,γUL​.

由均衡导频残差估计 σ^eq2\hat\sigma_\mathrm{eq}^2σ^eq2​，再按 QPSK 星座距离计算 LLR。软信息依次经过解交织、解扰和 LDPC 解码，恢复 UE 发送的信息比特。该过程与下行采用相同的软判决原则，但信道估计、频偏和噪声方差均来自上行自身的参考资源。

与 eRTM 的关系

Section titled “与 eRTM 的关系”
