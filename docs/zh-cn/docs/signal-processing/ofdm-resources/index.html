感知资源

Section titled “感知资源”

BS 单站感知使用下行已知符号集合

Ωsens⊆{(n,m):m∈SDL}.\Omega_\mathrm{sens} \subseteq \{(n,m):m\in\mathcal S_\mathrm{DL}\}.Ωsens​⊆{(n,m):m∈SDL​}.

同步、导频、通信数据和无载荷时的随机 QPSK 都可以参与感知，只要接收端能够获得对应的发送符号。恒模 ZC/QPSK 使逐元素去调制不会因符号幅度过小而显著放大噪声。若只选择部分感知资源，应保持慢时间采样间隔已知；均匀间隔资源可直接使用 Doppler FFT，非均匀资源则需要按实际采样时刻处理。

分辨率与无模糊范围

Section titled “分辨率与无模糊范围”

使用连续 NNN 个子载波时，基本时延分辨率为

Δτ=1B.\Delta\tau=\frac{1}{B}.Δτ=B1​.

因此单站距离分辨率为 Δrmono=c/(2B)\Delta r_\mathrm{mono}=c/(2B)Δrmono​=c/(2B)，以 LoS 为参考的双站超额路径分辨率为 Δdbi=c/B\Delta d_\mathrm{bi}=c/BΔdbi​=c/B。循环前缀给出无符号间干扰的最大时延扩展边界；更长的有效带宽改善时延分辨率，而更长的相干慢时间观测改善多普勒分辨率。
