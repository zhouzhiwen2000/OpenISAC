信号模型

当前 OpenISAC 系统包含一条 BS→UE 下行、一条 UE→BS 上行、RRR 个 BS 单站感知接收通道，以及一个 UE 双站感知观测。BS 的下行波形同时承担通信和照射信号；UE 接收同一波形并同时用于通信解调与双站感知。

双向通信信道

Section titled “双向通信信道”

令 x∈{DL,UL}x\in\{\mathrm{DL},\mathrm{UL}\}x∈{DL,UL} 表示下行或上行。链路 xxx 的时变多径信道写为

hx(t,τ)=∑l=1Lxαlxδ ⁣(τ−τlx−τdx(t))ej2π(fD,lx+Δfcx)t.h_x(t,\tau)= \sum_{l=1}^{L_x} \alpha_l^x \delta\!\left(\tau-\tau_l^x-\tau_d^x(t)\right) e^{j2\pi(f_{D,l}^x+\Delta f_c^x)t}.hx​(t,τ)=l=1∑Lx​​αlx​δ(τ−τlx​−τdx​(t))ej2π(fD,lx​+Δfcx​)t.

其中，LxL_xLx​ 是可分辨路径数，αlx\alpha_l^xαlx​、τlx\tau_l^xτlx​ 和 fD,lxf_{D,l}^xfD,lx​ 分别是第 lll 条路径的复系数、传播时延和多普勒频移；τdx(t)\tau_d^x(t)τdx​(t) 是两端参考时间不一致造成的链路定时偏移，Δfcx\Delta f_c^xΔfcx​ 是载波频偏。

下行与上行分别满足

yUEDL(t)=∫hDL(t,τ)sDL(t−τ) dτ+zUE(t),y_\mathrm{UE}^\mathrm{DL}(t) =\int h_\mathrm{DL}(t,\tau)s_\mathrm{DL}(t-\tau)\,d\tau +z_\mathrm{UE}(t),yUEDL​(t)=∫hDL​(t,τ)sDL​(t−τ)dτ+zUE​(t), yBSUL(t)=∫hUL(t,τ)sUL(t−τ) dτ+zBSUL(t).y_\mathrm{BS}^\mathrm{UL}(t) =\int h_\mathrm{UL}(t,\tau)s_\mathrm{UL}(t-\tau)\,d\tau +z_\mathrm{BS}^\mathrm{UL}(t).yBSUL​(t)=∫hUL​(t,τ)sUL​(t−τ)dτ+zBSUL​(t).

这里不要求 hDL=hULh_\mathrm{DL}=h_\mathrm{UL}hDL​=hUL​。TDD 在信道相干时间内可以近似利用传播互易性，但两端定时、频率偏移和收发响应仍需分别估计；FDD 的两个载波则直接视为两条独立频率响应。

BS 多通道单站信道

Section titled “BS 多通道单站信道”

将 RRR 个 BS 感知通道视为均匀线阵（ULA）。阵元间距为 dad_ada​，下行载波波长为 λ=c/fc\lambda=c/f_cλ=c/fc​，入射角 θ\thetaθ 从阵列法向（broadside）计量。阵列导向矢量定义为

UE 双站几何

Section titled “UE 双站几何”

对经过第 lll 个散射体的 BS→UE 路径，设 BS 到散射体、散射体到 UE、BS 到 UE 的距离分别为 dB,ld_{B,l}dB,l​、dl,Ud_{l,U}dl,U​ 与 dB,Ud_{B,U}dB,U​，则

τlbi=dB,l+dl,Uc,τLoS=dB,Uc.\tau_l^\mathrm{bi}=\frac{d_{B,l}+d_{l,U}}{c}, \qquad \tau_\mathrm{LoS}=\frac{d_{B,U}}{c}.τlbi​=cdB,l​+dl,U​​,τLoS​=cdB,U​​.

以直达径为定时参考后，双站感知观测的是超额时延

Δτlbi=τlbi−τLoS,Δdlbi=cΔτlbi.\Delta\tau_l^\mathrm{bi} =\tau_l^\mathrm{bi}-\tau_\mathrm{LoS}, \qquad \Delta d_l^\mathrm{bi}=c\Delta\tau_l^\mathrm{bi}.Δτlbi​=τlbi​−τLoS​,Δdlbi​=cΔτlbi​.

因此一个固定双站时延对应以 BS 和 UE 为两个焦点的椭圆，而不是单站模型中的圆。低多普勒分量通常归入杂波，动态多径则作为双站散射体。

采样时钟偏差

Section titled “采样时钟偏差”

名义采样间隔为 Ts=1/BT_s=1/BTs​=1/B。若接收端的实际采样间隔为 Ts,q=Ts−ΔTs,qT_{s,q}=T_s-\Delta T_{s,q}Ts,q​=Ts​−ΔTs,q​，则第 lll 条通信路径的离散样本可近似写为

yq,l[k]≈αls ⁣(kTs−τl−τd[0]−kΔTs,q)ej2π(fD,l+Δfc)kTs.y_{q,l}[k]\approx \alpha_l s\!\left(kT_s-\tau_l-\tau_d[0]-k\Delta T_{s,q}\right) e^{j2\pi(f_{D,l}+\Delta f_c)kT_s}.yq,l​[k]≈αl​s(kTs​−τl​−τd​[0]−kΔTs,q​)ej2π(fD,l​+Δfc​)kTs​.

该式忽略了 (fD,l+Δfc)ΔTs,q(f_{D,l}+\Delta f_c)\Delta T_{s,q}(fD,l​+Δfc​)ΔTs,q​ 的二阶小量。它揭示了三种可分离的影响：固定定时偏移移动帧起点，CFO 产生随时间累积的公共相位，SFO 则同时造成随时间增长的时偏和与子载波频率相关的相位斜率。
