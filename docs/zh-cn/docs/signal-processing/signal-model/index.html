信号模型

OpenISAC 使用同一套连续 OFDM 波形支持下行通信、BS 侧单站感知和 UE 侧双站感知。当前公开平台是 SISO 原型：一个 BS 发射通道、一个 BS 感知接收通道和一个 UE 接收通道。

BS 连续发送 OFDM 帧：

s(t)=∑γ=0∞sγ(t−γTF)s(t)=\sum_{\gamma=0}^{\infty}s_\gamma(t-\gamma T_F)s(t)=γ=0∑∞​sγ​(t−γTF​)

第 γ\gammaγ 帧的基带波形为：

BS-UE 信道

Section titled “BS-UE 信道”

UE 观察到的下行通信信道也是双站感知信道：

hUE(t,τ)=∑l=1Lαlδ(τ−τl−τd)ej2π(fD,l+Δfc)th_\mathrm{UE}(t,\tau) = \sum_{l=1}^{L} \alpha_l \delta(\tau-\tau_l-\tau_d) e^{j2\pi(f_{D,l}+\Delta f_c)t}hUE​(t,τ)=l=1∑L​αl​δ(τ−τl​−τd​)ej2π(fD,l​+Δfc​)t

其中 αl\alpha_lαl​、τl\tau_lτl​、fD,lf_{D,l}fD,l​ 分别为第 lll 条路径的复系数、时延和多普勒频移。τd\tau_dτd​ 和 Δfc\Delta f_cΔfc​ 表示 BS-UE 时序偏移和载波频偏。

在双站感知中，低多普勒或零多普勒分量被视为 clutter，动态多径分量被解释为有效散射体。

Section titled “单站信道”

BS 侧单站感知信道单独建模为：

hBS(t,τ)=∑p=1P+Cβpδ(τ−τs,p)ej2πfD,s,pth_\mathrm{BS}(t,\tau) = \sum_{p=1}^{P+C} \beta_p \delta(\tau-\tau_{s,p}) e^{j2\pi f_{D,s,p}t}hBS​(t,τ)=p=1∑P+C​βp​δ(τ−τs,p​)ej2πfD,s,p​t
