Why OpenISAC

面向高带宽 OTA 实验的 OFDM ISAC 平台

OpenISAC 适合需要真实空口实验的研究团队，在尽量保持代码可读、易改和贴近 PHY 的前提下，提供高带宽通信与感知能力。

为 OTA 实验而不是纯仿真设计

OpenISAC 处在“仿真代码”和“完整标准栈”之间，让研究者更快把想法跑到真实 USRP 链路上。

实时链路与前端工具分工清晰

C++ 负责实时 PHY 与感知主链路，Python 负责可视化、分析和网页配置等外围工具。

运行配置成本低

YAML 驱动运行时配置，仓库内已有 X310/B210 模板，同时提供 CPU 隔离和网页配置工具。
